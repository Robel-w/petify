# 🐾 Petify – Online Pet Marketplace

## 📌 Overview
**Petify** is a web-based pet marketplace platform that connects **pet sellers and buyers** in a secure and user-friendly environment. The system allows users to browse pet listings, communicate with sellers, and manage listings efficiently.

This project demonstrates key concepts in **full-stack web development**, including UI design, backend logic, and database management.

---

## 🎯 Purpose
Petify aims to solve:
- Lack of **trusted platforms** for buying pets  
- Difficulty finding **verified sellers**  
- No centralized system to **browse, compare, and contact sellers**  

---

## 👥 User Roles

### 🧑 Buyer
- Browse and search for pets  
- View pet details  
- Contact sellers  
- Leave reviews  

### 🏪 Seller
- Create and manage pet listings  
- Respond to buyer messages  
- Update or remove listings  

### 🛠️ Admin
- Approve or reject sellers  
- Monitor listings  
- Manage users and reports  

---

- 🔐 User Authentication (Login/Register)

## 🔄 System Workflow
1. User registers or logs in  
2. Seller posts pet listings  
3. Buyer searches and views pets  
4. Buyer contacts seller  
5. Transaction happens outside the platform  
6. Buyer leaves a review  

---

## ⚠️ Limitations
- No integrated payment system  
- Transactions handled outside the platform  
- Requires admin moderation
